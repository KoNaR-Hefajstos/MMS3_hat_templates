# Zestaw template'ów kompatybilnych z [płytką MMS2](https://github.com/KoNarRobotics/MMS2-F405)

### Small
Rozmiar płytki taki sam jak MMS2.
Bez dodatkowych otworów montażowych

### Full Size
Domyślny rozmiar hat'a czyli 100x100mm.
Dodatkowe otwory montażowe o rozstawie 90mm

### Extended
Płytka rozszerzona w jedną strone. Rozmiar 100x140mm ale w wypadku potrzeby można ją dalej powiększać zachowując wymiar 100mm stały.
Otwory montażowe takie same jak dla *Full Size*
